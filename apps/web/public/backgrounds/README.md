# Preset room backgrounds

Drop room background images here. Any filename works (they're listed
alphabetically); use `.png`, `.jpg`, `.jpeg`, or `.webp`.

- Recommended: portrait orientation, ~1080×1920, since rooms are tall/phone-shaped.
- The room dims them behind a dark overlay for text legibility, so richly
  colored images are fine.
- Suggested names for ordering: `bg01.jpg`, `bg02.webp`, `bg03.png`, …

The host picks a background from the room menu (gear → Change background).
After adding/removing files, restart the server so it re-scans (or it re-scans
per request already — a page refresh of the picker is enough).
